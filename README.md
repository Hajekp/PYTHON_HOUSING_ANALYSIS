# SoCal Housing Market Analysis (Python)

## Overview
This project analyzes the Southern California real estate market using Python and Pandas.
The goal was to replicate and expand on a SQL analysis to explore price trends across cities, bedroom counts, and price tiers.

## Dataset
- **Source:** Kaggle — SoCal Housing Dataset
- **Rows:** 15,474 listings
- **Columns:** street, city, bedrooms, bathrooms, square footage, price

## Questions Analyzed
1. What is the average, max, and min home price?
2. Which cities have the highest average home prices?
3. Which cities have the most listings?
4. How does average price change by bedroom count?
5. How are listings distributed across price tiers?

## Key Findings
- Newport Coast has the highest average home price at $1.8M
- San Diego has the most listings with 707, nearly double Los Angeles at 388
- Mid-range homes ($500k-$800k) make up the largest share at 6,214 listings
- Average price increases consistently with bedroom count up to 8 bedrooms
